Spatial query of coordinate reference systems
=============================================

In this workshop, I will start with discussing common problems that novice geographic information system (GIS) users face when trying to select a coordinate reference system (CRS) for their study.
I will then introduce `ProjPicker <https://github.com/HuidaeCho/projpicker>`_ (Projection Picker).
